--- title: Reporting Bugs ---

We use our GitHub issue tackers to handle bug reports, but we don't accept bug reports submitted through anything else. Do NOT email me about bugs!

If you're just reporting a typo on this website, you can simply make a Pull Request on the Wolfram website repository that fixes the typo. That's not possible with typos inside the client, though. You will need to make a bug report for those.

If you are reporting a crash, always include the crash report file! Crash reports are completely useless without that file, so if you don't include it, we will ban your account.

Here's a list of our Wolfram-related GitHub repositories. Find the one that's right for your bug report, read through the rules and then open an issue there. You will also need a GitHub account if you don't have one already.
